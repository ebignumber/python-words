WORD COMMANDS

a: Add a word to the current puzzle
s: Select a word to move
d: Delete a word from the current puzzle

MOVEMENT COMMANDS

Note: all movement commands (except for r) can have a number behind it so that the word moves more than one space. For example, running 2l moves the selected word two spaces to the right and 8k moves the word eight spaces upwards.

h: Move selected word to the left
j: Move selected word downwards
k: Move selected word upwards
l: Move selected word to the right
r: Rotate selected word

SERIES COMMANDS

w: Save the current puzzle in a new json file
W: Overwrite a puzzle in the series
x: Delete a puzzle in the series
c: Change series
o: Load data from a puzzle
m: Move a puzzle to a different location

MISCELLANEOUS

q: Quit program
?: Open this help
